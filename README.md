# Cómo visualizar los componentes

Los archivos de cada componente están importandos con la flag `type="module"`, esto para que las reglas, funciones y estilos pertenezcan a cada componente y no se mezclen. Pero al abrir el archivo index con doble click, es posible que ningún componente se note ya que navegador bloquea los archivos declarados como módulos. Para esto, se recomienda correr el archivo con un live server (extensión de VS Code Live Server y luego click derecho y seleccionar opción Open with Live Sever 😊)
